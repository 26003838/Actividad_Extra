¿Por qué elegí la licencia MIT?
Elegí la licencia MIT porque es una de las más sencillas y flexibles que existen. Permite que otras personas vean y usen mi código con total libertad, siempre y cuando incluyan el aviso de copyright original. Como este es un proyecto educativo, me parece la mejor opción para compartir lo aprendido sin restricciones complicadas.
